Web developer vs web designer

Many people use the words web designer and web developer interchangeably. Let me add a little clarity on the difference.

The web designer specialises more on the look, design and the layout of the website and it's user experience. He uses his skills to attract, inform and persuade clients. He is adept in graphics software programs like Photoshop, where he enhances, manipulates images and text. The web developer on the other hand translates these designs into code which the computer can understand. He deploys the website into a hosting company and registers it with a search engine and optimizes the website for search engine search (SEO). The person who can design a webpage but cannot write or understand code is a graphic designer.
