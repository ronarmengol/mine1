Responsive design

When sizing block elements, use relative sizing and not absolute sizing. Use percentages and viewport sizing (vh and vw) and not pixels and the metric system.

If you use pixels (px) to size a div width, use for example 'max-width: 500px;'. The div will overflow for smaller viewports, but the max-width is responsive to the viewport being sized down.

In the HTML tag in the CSS use 'overflow-x: hidden', so the default display will not be scrolling left and right.
